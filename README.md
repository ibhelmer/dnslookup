# DNS Lookup Tool
Forfatter: Ib Helmer Nielsen / UCN

 Beskrivelse:
   Dette program udfører et DNS-opslag på et hostnavn og
   viser IPv4-, IPv6- og reverse DNS-information. Programmet 
   er skrevet i python3

 Syntaks:
   python3 dns_lookup.py <hostnavn>
   eller
   ./dns_lookup.py <hostnavn>

 Eksempel:
   python3 dns_lookup.py www.ibhelmer.dk
